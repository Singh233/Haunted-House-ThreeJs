## Run Project Locally

Clone the project

```bash
  git clone https://github.com/Singh233/Haunted-House-ThreeJs.git
```

Go to the project directory

```bash
  cd Haunted-House-ThreeJs
```

Install the packages

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
